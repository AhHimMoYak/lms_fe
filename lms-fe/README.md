#  LMS project FE

## Getting Started

### INIT

```bash
git clone {{ repository_url }}

cd {project}
npm install
npm run dev
```


### react 구조
```
|__public
|__src
    |____assets
    |____authentication
    |____components
    |____hooks
    |____pages
    |____styles

```


### 폴더별 용도
- public : 정적 파일 보관
- assets : 이미지 및 폰트
- authentication : 인증, 인가 관리
- components : 재사용 가능 컴포넌트 
- hooks : 커스텀 훅
- pages : 라우팅 적용 페이지
- styles : css 파일
